# Stylesheet for MacDown

## Installation

Download and copy the CSS files in `~/Library/Application Support/MacDown/Styles`

## Usage

- resume-clean:

  ```
  # Name
  Contact info 1 | info 2

  **About**
  Some text

  ## Experiences
  **Job Title**
  *Company*, Location [Date Range]
  Description

  | Languages | Skills | Other |
  |:----------|:-------|:------|
  | text      | text   | text  |
  | text      | text   | text  |
  ```
